🚀 Get Started with LibreAIChat Community Edition

LibreAIChat Community Edition is a fully open version of the app you see on libreaichat.com — minus billing, rate limits, and login.

It’s designed for developers, self-hosters, educators, or anyone who wants a plug-and-play local AI chat interface.

View GitHub Repository →

🔧 What's Included

  • 🧠 Full frontend + Go Fiber backend
  • 🧱 Docker Compose setup (Ollama + PostgreSQL + Caddy)
  • 🎨 Clean UI for chat with model selector
  • 📦 Pre-configured to auto-download supported models
  • 🔓 Includes GitHub login, no billing or subscription logic
